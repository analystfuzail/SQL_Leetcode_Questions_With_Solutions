## 🛠️ How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/analystfuzail/SQL_Leetcode_Questions_With_Solutions.git
   ```

2. Open any `.sql` file — the problem statement is at the top as a comment, solution follows below.

3. Run the solution in **MySQL Workbench** or directly on [LeetCode](https://leetcode.com/problemset/database/).
